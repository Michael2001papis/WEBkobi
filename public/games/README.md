# תיקיית משחקים

## איך להוסיף משחק חדש (Plug & Play):

1. **צור תיקיה חדשה** עם שם המשחק (למשל: `my-game`)
2. **הכנס את כל קבצי המשחק** לתיקיה הזו
3. **ודא שיש קובץ כניסה ראשי** (`index.html`)
4. **עדכן את `games.json`** והוסף את המשחק לרשימה:

```json
{
  "games": [
    {
      "id": "my-game",
      "title": "שם המשחק",
      "description": "תיאור קצר של המשחק",
      "folder": "my-game",
      "badges": ["new", "fast"]
    }
  ]
}
```

**Badges זמינים:**
- `"new"` - משחק חדש
- `"classic"` - משחק קלאסי
- `"fast"` - משחק מהיר

**שימו לב:** אין צורך לעדכן קוד! רק JSON + תיקיית משחק.

## מבנה תיקיה מומלץ:

```
public/games/
  └── my-game/
      ├── index.html (או game.html)
      ├── thumbnail.png (אופציונלי)
      ├── js/
      ├── css/
      └── assets/
```

## הערות:

- המשחקים נפתחים בחלון חדש (`target="_blank"`)
- כל משחק צריך להיות עצמאי (standalone)
- ודא שהמשחק עובד גם ב-RTL אם רלוונטי
